## 🛠️ Desarrollo local

Clona el repo y abre directamente en el navegador:

```bash
git clone https://github.com/oscarherruzo/OscarHerruzo-Web.git
cd OscarHerruzo-Web
# Abre index.html en tu navegador
```

O con Live Server en VS Code:
1. Instala la extensión **Live Server**
2. Clic derecho sobre `index.html` → **"Open with Live Server"**

---

## 📄 CV descargable

El fichero `cv.html` es un currículum en formato A4 simulado, imprimible directamente desde el navegador:

1. Abre `cv.html`
2. Clic en **"Imprimir / Guardar PDF"**
3. En el diálogo del navegador selecciona **"Guardar como PDF"**

---

## 🔄 Despliegue

El sitio se despliega automáticamente en **GitHub Pages** con cada `push` a `main`:

```bash
git add .
git commit -m "descripción del cambio"
git push
```

Los cambios están en producción en **1-2 minutos**.

**Configuración de Pages:**
- Source: `Deploy from a branch`
- Branch: `main` / `(root)`
